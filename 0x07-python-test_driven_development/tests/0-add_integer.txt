Numbers
========
add_integer() return the sum of its arguments.

 >>> add_integer = __import__('0-add_integer').add_integer
 >>> add_integer(2, 3)
 5
 >>> add_integer(2.3, 2.7)
 4
 >>> add_integer(-1, 1)
 0
 >>> add_integer(4)
 102
